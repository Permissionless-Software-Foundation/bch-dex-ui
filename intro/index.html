BCH DEX

Read Before Entering:

The BCH DEX is a decentralized network for trading SLP tokens on the Bitcoin Cash blockchain. Clicking the button below will take you to a read-only demo. You can not trade tokens, but can only view the tokens listed for sale on the network.

Optional Information:

The BCH DEX is still in its prototyping phase. The code is rough and prone to bugs. Keep this in mind, and check your expectations.

The DEX is not intended to be operated as a typical '.com' website. It's intended to be operated by individuals, in the comfort of their own home, on a standard computer or Raspberry Pi. To start trading and to find out more, check out the bch-dex GitHub repository.
